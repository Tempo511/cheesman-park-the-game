# Cheesman Park: The Game

**Play it: [cheesmanparkthegame.com](https://www.cheesmanparkthegame.com)** — free, in your browser, no app needed. Phones play in landscape.

A pixel-art homage to Denver's Cheesman Park and its real history. The park sits
on Mount Prospect, Denver's first cemetery (1858) — and the city's 1893 grave
clearing, botched by the undertaker E.P. McGovern, was never finished. By day you
stroll the park, find real landmarks, feed squirrels, and pet Sprout. By night,
the former residents claw out of the Great Lawn. Every fifth night, McGovern
himself shows up. Beat him and the Denver Botanic Gardens opens for a bonus run.

Made with ❤️ by **Casey Cantor**, a few blocks from the park, with Claude as
programming partner. Every sprite and sound is procedural — there are no asset
files. The simulation is deterministic and replayable, which powers the
leaderboard's anti-cheat and, someday, co-op.

Contact: hi@cheesmanparkthegame.com

## Run it locally

```
python3 serve.py   # serves on http://localhost:8123
npm test           # headless sim + DOM smoke tests
```
